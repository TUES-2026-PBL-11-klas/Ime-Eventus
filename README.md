# Eventus

**School Event and Activity Coordination Hub**

Eventus is a comprehensive platform for managing school events, activities, and resources. It provides role-based access for administrators, teachers, students, and coordinators to streamline event organization, room booking, and student participation tracking.

## Features

- **Event Management** - Create, edit, and categorize school events (competitions, workshops, clubs, celebrations, volunteer activities)
- **Approval Workflow** - Multi-level approval process for event proposals
- **Room & Resource Booking** - Manage room availability and detect scheduling conflicts
- **Registration System** - Student registration with capacity tracking and waitlists
- **Role-Based Access** - Different views and permissions for admins, teachers, students, and coordinators
- **Reports & Analytics** - Track attendance, participation, and generate insights
